Languages

What should you learn first?

A programming language is a tool built for a specific purpose. For most beginners, HTML is the right place to start. Below is what five of the most common beginner languages look like.

HTML structures the content of every webpage. It is not technically a programming language, but it is where every web developer begins.

<!-- A comment. The browser ignores this. -->
<h1>Hello, World!</h1>
<p>This is a paragraph of text.</p>
<a href="https://example.com">This is a link</a>
<img src="photo.jpg" alt="A description">
Start here if you want to build websites and have never written a line of code. You will see results in minutes.

CSS controls how HTML looks: colors, fonts, spacing, layout. It is what separates a raw document from a designed page.

/* Change how the whole page looks */
body {
  background-color: #faf7f2;
  font-family:      Georgia, serif;
  color:            #1a1612;
  max-width:        700px;
  margin:           0 auto;
}
h1 { font-size: 2.5rem; }
Start here if you are drawn to design. CSS is immediately gratifying: one line changes the entire appearance of a page.

JavaScript makes websites interactive. Buttons that respond, forms that validate, animations, games — all of it is JavaScript.

// Store a value in a variable
let name = "World";

// Print to the browser's console
console.log(`Hello, ${name}!`);

// React when a button is clicked
document.querySelector("button")
  .addEventListener("click", () => {
    alert("You clicked it!");
  });
Start here if you already know basic HTML and CSS and want to add real interactivity to your pages.

Python reads almost like plain English. It is the dominant language for data science, AI, and automation.

# Print a message to the terminal
print("Hello, World!")

# Define a reusable function
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
Start here if you are interested in data, AI, science, or automation rather than building websites.

Swift is Apple's language for building iPhone, iPad, and Mac apps. It is modern, fast, and designed to be readable.

// Swift — used for iOS and macOS apps
import Foundation

let greeting: String = "Hello, World!"
print(greeting)

func greet(name: String) -> String {
    return "Hello, \(name)!"
}
Start here if you own a Mac and want to build iPhone or iPad apps. You will need Xcode, which is also free.

Learning roadmap

A clear path from zero to building things.

The most common mistake beginners make is trying to learn everything at once. Follow this sequence.

1
Weeks 1–2  ·  ~8 hours total

HTML Fundamentals

Learn what tags are and how web pages are structured. Build your first page. Small scope, high confidence.

2
Weeks 2–4  ·  ~12 hours total

CSS Fundamentals

Style your HTML. Colors, fonts, padding, borders. This is where things start looking like real websites.

3
Month 2  ·  ~20 hours total

JavaScript Basics

Variables, functions, loops. Then DOM manipulation: making buttons respond and handling user input.

4
Month 2–3  ·  ~6 hours total

Git and GitHub

Save your work. Share your projects publicly. Host websites for free. Essential for professional growth.

5
Month 3 onward

Build Projects

This is where real learning happens. Pick something small. Build it. Show it to someone. Repeat indefinitely.

How this page works

How the dark mode on this page was built.

The toggle is roughly twenty lines of CSS and one small JavaScript function. Here is how you can add it to any site.

1

Define CSS variables

Define variables inside :root. Every element referencing var(--bg) will update automatically. 

:root {
  --bg:     #faf7f2;
  --text:   #1a1612;
}
body {
  background: var(--bg);
}
2

Override for dark mode

When the <html> element has data-theme="dark", redefine those variables. 

[data-theme="dark"] {
  --bg:     #100e0b;
  --text:   #ede6da;
}
3

Toggle with JavaScript

function toggleTheme() {
  const html   = document.documentElement;
  const isDark = html.dataset.theme === "dark";
  html.dataset.theme = isDark ? "light" : "dark";
}

Publishing your work

How to put a page online, for free.

1

GitHub Account

Go to github.com and sign up.

2

New Repository

Click +, then "New repository." Name it my-site.

3

Upload index.html

Inside the repo, click "Add file" and drag your index.html in.

4

Enable Pages

In Settings > Pages, select main branch and click Save.

5

Site is live

Your site will be at yourusername.github.io/my-site.
